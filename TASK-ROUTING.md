# 个性化任务路由规则

## 一、任务选择顺序

1. 判断用户路径：家庭、学生、教师；
2. 判断学段；
3. 判断内容领域；
4. 确定一个主要诊断维度；
5. 区分知识缺口、结构断点或执行断点；
6. 选择任务族；
7. 设定完成证据；
8. 设定升级、保持和退阶。

不得从“知识点名称”直接跳到“十道同类题”。

## 二、任务族

| 任务族 | 适用诊断维度 | 典型产出 |
|---|---|---|
| A 对象辨认 | D1 | 标注、分类、范围说明 |
| U 单位参照 | D2 | 单位卡、参照说明、量纲检查 |
| C 概念意义 | D3 | 例子反例、定义改写、意义解释 |
| R 关系结构 | D4 | 关系句、结构图、中间目标 |
| S 表示转换 | D5 | 实物—图—表—式—图像互译 |
| O 运算变换 | D6 | 依据标注、错误节点修复、等价链 |
| P 推理论证 | D7 | 条件结论表、证明链、反例、分类 |
| M 建模应用 | D8 | 变量表、假设、模型、检验 |
| T 迁移生成 | D9 | 变式、不变量、创编、推广 |
| E 监控执行 | D10 | 计划卡、检查表、时间记录 |

## 三、时长与负担

- G1—G2：8—15 分钟，优先口头、实物和图；
- G3—G6：15—25 分钟，最多一个核心关系；
- G7—G9：20—35 分钟，最多一个变换链或证明断点；
- G10—G12：25—45 分钟，最多一个主结构和一个边界检查。

## 四、提示层级

### 0级：不提示

让学生先呈现真实起点。

### 1级：目标提示

“题目最终要确定什么？”

### 2级：对象或关系提示

“哪两个量、式、图形或事件之间有直接关系？”

### 3级：表示提示

“能否画图、列表、设变量或写出一个中间式？”

### 4级：半成品结构

提供部分图、表、等式或证明框架。

### 5级：示范

完整示范一题，再要求学生完成结构相同但表面不同的迁移题。

智能体必须记录使用到哪一级提示。

## 五、升级规则

连续两次满足以下多数条件才升级：

- 独立完成；
- 能解释依据；
- 能处理一种简单变化；
- 能主动检查或修正；
- 所用提示不高于1级。

一次只增加一个变量：

- 对象数量；
- 条件数量；
- 符号抽象程度；
- 表示方式；
- 推理链长度；
- 参数或分类；
- 开放程度；
- 时间限制。

## 六、保持规则

基本理解但不稳定时：

- 保持核心目标；
- 更换表面情境；
- 只改变一个非核心变量；
- 降低成人提示；
- 增加解释或检查要求。

## 七、退阶规则

出现以下情况时退阶：

- 无法复述任务目标；
- 多个对象、条件或符号同时混淆；
- 成人或智能体完成了大部分关键步骤；
- 连续两次无法启动；
- 情绪明显受挫。

退阶方式：

- 减少对象和条件；
- 从符号退回图、表、操作或具体例子；
- 把完整证明拆为一个中间结论；
- 把开放问题改为选择与解释；
- 去掉时间限制；
- 提供半成品结构。

## 八、禁止任务

- 一次同时训练多个主要卡点；
- 只因为做错就布置大量同类题；
- 只换数字冒充迁移；
- 不规定成果证据；
- 让家长替孩子抄标准答案；
- 未经校验自由生成高风险考试题答案；
- 用超纲难题证明学生“能力不足”。
