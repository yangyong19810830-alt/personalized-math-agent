# 跨学段示例

## 示例1：一年级关键词依赖

### 输入

“池塘里有5只鸭子，又游来3只。孩子看到‘又’就写5−3。”

### 合格诊断

- 主要卡点：数量变化动作尚未与算式稳定对应；
- 维度：D3、D4、D5；
- 证据：学生根据词选择符号，没有说明数量是增加还是减少；
- 任务：L02做动作再写算式；
- 暂不训练：口算速度。

## 示例2：四年级单位换算

### 输入

知道1米=100厘米，却不能解释3米为什么是300厘米。

### 合格诊断

- 主要卡点：单位改变与单位个数变化尚未建立；
- 任务：P07单位改变、数量不变；
- 完成标准：能说明厘米单位更小，同一长度包含的单位个数更多。

## 示例3：六年级分数

### 输入

“甲的2/3和乙的2/3一样多。”

### 合格诊断

- 主要卡点：单位“1”被忽略；
- 任务：P08分数的单位“1”；
- 迁移：给出单位“1”相同与不同的两组情境。

## 示例4：七年级移项

### 输入

学生说“移到另一边就变号”，在含括号时频繁出错。

### 合格诊断

- 主要卡点：等价变换依据被口诀替代；
- 任务：J03等价变形依据标注；
- 暂不训练：大量重复方程。

## 示例5：八年级几何证明

### 输入

学生在证明DE=DF时，第一步直接写“因为DE=DF”。

### 合格诊断

- 主要卡点：条件与待证结论未区分；
- 任务：J09几何条件—结论表，再进入J11证明链补全。

## 示例6：九年级二次函数

### 输入

会求顶点，但参数改变后不知道图像怎样变化。

### 合格诊断

- 主要卡点：公式计算与参数作用脱节；
- 任务：J08参数改变会怎样；
- 完成标准：一次只改变一个参数，预测并验证图像变化。

## 示例7：高一函数定义域

### 输入

化简正确，但解题中遗漏根号和分母限制。

### 合格诊断

- 类型：执行断点兼结构断点；
- 任务：H01定义域与约束先行；
- 检查点：每次变形后重新核对约束是否保持。

## 示例8：高二数列

### 输入

看到递推式就尝试套等差、等比公式。

### 合格诊断

- 主要卡点：未识别“当前状态怎样生成下一状态”的结构；
- 任务：H07数列递推结构图。

## 示例9：高三导数

### 输入

求出f′(x)后继续大量计算，不知道要判断什么。

### 合格诊断

- 主要卡点：局部运算与整体目标断开；
- 任务：H02局部步骤与整体目标＋H05导数结果回到函数变化，但本轮先做H02。

## 示例10：教师课堂改造

### 输入

教师准备先讲线段计数公式，再让学生练习。

### 合格改造

先给4、5、6个点，让学生形成“每条线段由两个端点决定”的关系；收集连线、列表和端点组合三类作品；讨论重复计算为什么除以2；最后改变点的排列验证结构是否保持。

## 示例11：证据不足

### 输入

家长说“孩子数学思维差”。

### 合格回应

不接受该标签作为诊断。请上传最近一道典型错题，并说明是独立完成还是得到过提示；若暂时无法上传，可先做一项10分钟验证任务，再形成低置信度判断。
